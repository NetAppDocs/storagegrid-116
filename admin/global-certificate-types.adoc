= Supported server certificate types
:icons: font
:imagesdir: ../media/

[.lead]
The StorageGRID system supports custom certificates encrypted with RSA or ECDSA (Elliptic Curve Digital Signature Algorithm).

For more information on how StorageGRID secures client connections for the REST API, see xref:../s3/index.adoc[Use S3] or xref:../swift/index.adoc[Use Swift].
